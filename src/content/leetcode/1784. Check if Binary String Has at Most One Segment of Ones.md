## 思路
題目有點難理解，總體來說，就是兩個 1 之間不能有 0 的存在。
## 程式碼
### 模擬
```cpp
class Solution {
public:
    bool checkOnesSegment(string s) {
        bool havezero = false;
        for(int i = 1; i < s.length(); i++) {
            if(havezero && s[i] == '1') return false;
            else if(s[i] == '0') havezero = true;
        }
        return true;
    }
};
```
## 複雜度分析
- 時間複雜度：$O(n)$
- 空間複雜度：$O(1)$
