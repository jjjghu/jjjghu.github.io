## 思路
對每個字串做以下操作：
1. 計算字元的 ascii 碼總和。
2. 將他模 26 之後取補數。
3. 把對應的字元加入答案中。
## 程式碼
### 模擬
```cpp
class Solution {
public:
    string mapWordWeights(vector<string>& words, vector<int>& weights) {
        string res;
        for(string& s : words) {
            int sum = 0;
            for(char& c : s) {
                sum += weights[c - 'a'];
            }
            sum = (25 - sum % 26);
            res += (char)('a' + sum);
        }
        return res;
    }
};
```
## 複雜度分析
- 時間複雜度：$O(nm)$，字元的數量。
- 空間複雜度：$O(n)$
