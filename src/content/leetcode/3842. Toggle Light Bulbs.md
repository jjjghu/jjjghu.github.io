## 思路
創建一個數組，模擬燈泡開關即可。\
不過如果燈泡的數量不只 100，而是 $10^9$ 這麼多，就需要改成計算那些數字的出現次數為奇數次了。
## 程式碼
### 模擬
```cpp
class Solution {
public:
    vector<int> toggleLightBulbs(vector<int>& bulbs) {
        const int MX = 101;
        vector<bool> state(MX, false);
        for(int x : bulbs) {
            state[x] = !state[x];
        }
        vector<int> res;
        for(int i = 0 ; i < MX; i++) {
            if(state[i]) {
                res.push_back(i);
            }
        }
        return res;
    }
};
```
## 複雜度分析
- 時間複雜度：$O(n)$
- 空間複雜度：$O(n)$
